# Introduction

Welcome to the osinfra.io platform documentation.

osinfra.io is an open-source, team-first Infrastructure as Code reference implementation built on Google Cloud Platform, OpenTofu, and Kubernetes.

## Platform Teams

The platform is organized into teams, each with a distinct role:

- **Logos** — Organizational structure: GCP folder hierarchy, Google Identity groups, GitHub teams and repositories, Datadog teams
- **Corpus** — Team infrastructure: GCP projects, shared VPC and networking, DNS zones, Artifact Registry, workload identity pools
- **Pneuma** — Kubernetes workloads: GKE clusters, cert-manager, Istio service mesh, Datadog monitoring, OPA Gatekeeper
- **Arche** — Reusable OpenTofu child modules consumed by all infrastructure layers
- **Ekklesia** — Platform documentation (this site)
- **Techne** — Shared tooling: reusable GitHub Actions workflows, pre-commit hooks, developer environments
