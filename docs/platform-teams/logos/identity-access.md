# Identity & Access

Logos manages centralized identity and access control for the platform. Google Identity groups map team boundaries to GCP IAM, ensuring access is granted to groups rather than individuals.

- **Google Identity groups**: Team-scoped groups that control access to GCP resources across all environments
- **User management**: Users are provisioned into the appropriate groups, with group membership managed as code in pt-logos
