# Coding Conventions

<figure><img src="../../../.gitbook/assets/opentofu.png" alt="" width="170"><figcaption></figcaption></figure>

Coding standards for this platform are maintained as GitHub Copilot workspace instructions in the [pt-ai-context](https://github.com/osinfra-io/pt-ai-context) repository. The canonical reference is:

[`.github/instructions/team.instructions.md`](https://github.com/osinfra-io/pt-ai-context/blob/main/.github/instructions/team.instructions.md)

These instructions cover file structure, ordering rules, formatting, naming conventions, module source pinning, lifecycle patterns, and more. They are the authoritative source for how code in this platform should be written.
